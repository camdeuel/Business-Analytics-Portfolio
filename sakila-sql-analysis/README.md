# Sakila Movie Rental Business Analysis

## Overview

This project uses SQL to analyze operational and financial data from the Sakila movie rental database. The objective was to answer business-focused questions related to inventory management, revenue generation, customer demographics, and store performance using relational database analysis.

## Business Questions

### 1. Which actors appeared in the most films?

Analyzed actor participation across the film catalog by joining actor, film_actor, and film tables to determine which actors appeared most frequently.

### 2. What is the inventory value of each store?

Evaluated the number of films held by each store location and calculated the total replacement cost of inventory.

### 3. Which movie categories generate the most revenue?

Measured total and average revenue by film category using rental and payment transaction data.

### 4. How does actual rental revenue compare to expected rental revenue?

Compared customer payments against expected rental charges to identify differences between projected and actual revenue.

### 5. Which countries have the largest customer bases outside the United States?

Analyzed customer distribution across international markets to identify regions with the strongest customer presence.
